Introduction

This directory contains the software examples for using the EBI on an AVR®32 AT32UC3A.

 

EBI description

The EBI is the External Bus Interface. This component allows to add several components to your AVR®32 AT32UC3A processor, such as memories, displays, and so on.

  • EBI examples for AVR32 AT32UC3A Series.

  • SDRAM Controller : A SDRAM controller that uses the EBI. With an interactive example of usage.
