Introduction

This directory contains the software example for using the SDRAM Controller through the EBI on an AVR®32 AT32UC3A.

 

SDRAMC description

The SDRAMC is the Synchronous Dynamic RAM Controller, which uses the EBI to connect external memories to the AVR®32 AT32UC3A.

  • SDRAMC examples for AVR®32 AT32UC3A Series.

  • a SDRAM Example : A SDRAM example based on a 32MB SDRAM from Micron Technology (MT48LC16M16A2TG7E).
