Introduction

This directory contains the hardware components drivers for all AVR®32 AT32UC3 Series.

Each software is provided with full source code, example of usage, rich html documentation and ready-to-use projects for the IAR EWAVR32 and GNU GCC compilers.

All examples are using the DRIVERS package that is part of the AVR32 UC3 series complete software framework.

 

COMPONENTS Directory Description

This directory consists of the following contents:

  • COMPONENT Drivers for AVR32 AT32UC3 Series.

  • DISPLAY directory : LCD drivers.

  • MEMORY directory : Memory drivers.
