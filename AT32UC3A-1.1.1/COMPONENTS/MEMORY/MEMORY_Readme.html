Introduction

This directory contains the drivers for memory device like LCD for all AVR®32 AT32UC3 Series.

 

COMPONENTS/MEMORY Directory Description

This directory consists of the following contents:

  • MEMORY Drivers for AVR32 AT32UC3 Series.

  • Dataflash: the Atmel AT45DB drivers (through SPI). The AT45DB642 memory is present on both EVK1100 and EVK1101 evaluation kits.

  • SDRAM Micron MT48LC16M16A2TG-7E driver (through SDRAM controller (EBI)): This SDRAM is present on the EVK1100 evaluation kit.
