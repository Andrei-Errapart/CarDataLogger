Introduction

This directory contains the hardware drivers for Display device like LCD for all AVR®32 AT32UC3 Series.

Each software is provided with full source code, example of usage, rich html documentation and ready-to-use projects for the IAR EWAVR32 and GNU GCC compilers.

 

COMPONENTS/DISPLAY Directory Description

This directory consists of the following contents:

  • DISPLAY Drivers for AVR32 AT32UC3 Series.

  • EA DIP204 : Driver example for LCD EA DIP204. This LCD is present on the EVK1100 evaluation kit.
