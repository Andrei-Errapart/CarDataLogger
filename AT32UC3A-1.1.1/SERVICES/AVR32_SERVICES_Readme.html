Introduction

This directory contains the software services for all AVR®32 AT32UC3 Series.

Each software services is provided with full source code, example of usage, rich html documentation and ready-to-use projects.

All examples are using the DRIVERS package that is part of the AVR32 UC3 series complete software framework.

 

SERVICES Directory Description

This directory consists of the following contents:

  • Services for AVR32 AT32UC3 Series.

  • DSPLib : a set of optimized digital signal processing functions, with an example of usage per function.

  • FAT12/16/32 File System : with two examples of usage.

  • freeRTOS.org Operating System Demo Application: an example with the AVR32 UC3 freeRTOS.org port.

  • A basic two-in-one web server and TFTP server demo: this demo is using both the lwIP TCP/IP stack and the AVR32 UC3 freeRTOS.org port.

  • lwIP TCP/IP stack: the lwIP TCP/IP stack and its port on the AVR®32 AT32UC3 Series micro-controllers that have a MACB controller.

  • Abstraction layer for memory interfaces: an example of usage with the AT45DBX Dataflash memory driver.

  • USB Bootloader using DFU (Device Firmware Upgrade) Class: with an ISP(In-System Programming) example.

  • USB Mass Storage Class: Device and Host Mass Storage class examples.

  • USB HID class: HID mouse device class example.

  • USB CDC class: CDC device class example.
